# 社团招新智能匹配平台（初版）

## 项目定位
面向高校社团招新场景，解决新生“不知道该报哪个社团、报名流程分散、信息不透明”的痛点。

## 已实现的产品结构
- 新生兴趣画像：输入兴趣方向、时间投入、社交偏好、成长目标、时间偏好。
- AI智能匹配：基于规则模型计算匹配度并给出 Top3 推荐。
- 社团详情查看：查看社团简介和标签特色。
- 一键报名：选定意向社团后直接报名，形成完整闭环。

## 页面模块
- 画像问卷区（采集偏好）
- 匹配结果区（推荐排序+推荐理由）
- 报名流程区（步骤说明+提交报名）

## 适配的业务痛点
1. **信息不对称**：通过推荐理由与社团标签让选择更可解释。
2. **流程繁琐**：将“发现-比较-报名”集中在同一页面。
3. **匹配效率低**：通过兴趣画像和目标偏好生成优先级推荐。

## 如何使用
1. 打开 `index.html`。
2. 在“新生兴趣画像”中填写偏好并点击“生成智能匹配”。
3. 在结果中查看详情并选择意向社团。
4. 点击“立即报名”完成模拟报名。

## 后续可优化方向

- 使用大模型进行自然语言兴趣解析和个性化推荐解释。
- 新增“学长学姐评价”“社团活动日历”“招新进度追踪”。
- 增加运营后台：社团端发布招新岗位、审核报名、查看转化漏斗。
